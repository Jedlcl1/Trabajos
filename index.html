Conceptos Básicos de Color CSS

Existen diferentes formatos de color en css, incluyendo nombres de colores html, HEX, RGB y HSL

Cómo utilizar los códigos Hex de color en CSS

Los codigos Hex o Hexadecimal de color, son el metodo mas común de agregar color a un elemento HTML con CSS. En la hoja de estilos, se puede utilizar la propiedad para cambiar el color de fondo de una pagina web

    
CSS

    
   /* En tu hoja de estilos .css */
   body { color : #008F39; }

Una segunda opción es incluirlos los estilos en el head de el documento HTML usando style

    
HTML

    
   /* En tu documento .html*/
   <head> 
     <style> 
       body { color: #008F39; } 
     <style> 
   <head>

Se puede utilizar la propiedad CSS color con un codigo Hex de cualquier elemento HTML, la etiqueta body fue tan solo un ejemplo

Cómo utilizar nombres de colores HTML en CSS

Los nombres de los colores HTML son otra forma de estilo al contenido en CSS y a veces puede ser mas facil entender. Puedes utilizar el nombre de un color de la misma forma que un codigo Hex de color

    
CSS

    
   /* En tu hoja de estilos .css */
   body { color : green; }

Actualmente son 140 los nombres de los colores que son compatibles en todos los navegadores modernos. Te dejo una lista con ellos Colores compatibles actualmente

Cómo utilizar los valores de los colores RGB en CSS

RGB, que significa Red, Green y Blue(Rojo, verde y Azul), es un sistema de color comunmente encontrados en muchas aplicaciones de diseño y tecnologias, y mas recientemente se ha convertido en una de los diseñadores web y programadores.
Estos se pueden usar de la siguiente manera: 

    
CSS

    
   /* En tu hoja de estilos .css */
   body { color : rgb(0, 143, 57);  }

Una de las ventajas de usar RGB en CSS, es la capacidad de controlar la opacidad ademas de los colores. Adición de una "a" en el RGB(), que permite un cuarto valor que se debe asignar que determina la transparencia del color en una escala del 0 al 1

    
CSS

    
   /* En tu hoja de estilos .css */
   body { color : rgba(0, 143, 57, 0.5);  }

Si deseas utilizar RGB pero no sabes como sacar un color, hay varias formas de saber como se conforma el color que quieres, la primera experimentando variaciones de tonalidades hasta llegar a el color que quieres, la segunda buscar el codigo de color en google, o la tercera y que se me hace mas practica es usar un selector de color RGB, para el cual te dejo uno selector de color RGB

Cómo utilizar los valores HSL color en CSS

HSL, significa Hue, Saturation, Lightness (Matiz, Saturación y Luminosidad), otro sistema de color usado en muchas aplicaciones. El tono es medido en grados de 0 a 360, mientras que la saturción y luminosidad usa una escala de 0% a 100%.
Este se implemente de la siguiente manera:

            
CSS

            
           /* En tu hoja de estilos .css */
           body { color : hsl(120, 100%, 25);  }

del mismo modo, HSL permite agregar la letra "a" para controlar la transparencia del color, su uso es identico que en RGB, cin un valor entre 0 y 1.

            
CSS

            
           /* En tu hoja de estilos .css */
           body { color : hsla(120, 100%, 25, 0.5);  }
